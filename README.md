# BiteBalance

BiteBalance is a web-based nutrition tracking system that helps users manage their diet and monitor their health.

## Features
- User signup and login
- Track daily food intake
- Generate personalized diet plans
- Monitor nutrition progress
- Email alerts and notifications

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- JavaScript
- Composer

## Installation

1. Clone the repository
git clone https://github.com/HarshaVardhini01/BiteBalance.git

2. Move project to htdocs
3. Start Apache and MySQL using XAMPP
4. Import database
5. Open browser and run project
